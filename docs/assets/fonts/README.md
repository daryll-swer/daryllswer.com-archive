# Self-Hosted Fonts

The generated GitHub Pages archive self-hosts the public Google Fonts used by
the canonical WordPress site:

- `Poppins` for body/content text.
- `Raleway` for headings/titles.

These files are WOFF2 webfont files fetched from the public Google Fonts CSS
endpoint recorded in `manifest.json`. The source font families are distributed
under the SIL Open Font License 1.1 in the Google Fonts collection.

Licence/provenance files:

- `OFL-Poppins.txt`
- `OFL-Raleway.txt`
- `manifest.json`

These font files are third-party artefacts. They are not covered by the
repository tooling MIT licence or by the mirrored article content licence.
