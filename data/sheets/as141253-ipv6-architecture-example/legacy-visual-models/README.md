# Legacy AS141253 Visual Models

These are historical design references for maintainers, AI/LLM agents, and software developers.

They are deliberately excluded from GitHub Pages. `../visual.html` is the sole supported public AS141253 visual model.

The generator functions remain in `scripts/ipv6_visual_options.py`; no public HTML pages are produced for these models.

## Archived Models

- `At-a-glance allocation`: generator function `allocation_path_section`.
- `Operational branches`: generator function `operational_branches_section`.
- `Purpose map`: generator function `purpose_map_section`.
- `Branch cards`: generator function `branch_cards`.
- `Purpose cluster graph`: generator function `purpose_cluster_graph`.
- `Visual-options gallery`: generator function `render_index`.
