# Escalation Rules

Stop and ask the owner before:

- Deleting, recreating, force-pushing, replacing the default branch, or changing
  visibility of any GitHub repository.
- Creating a private GitHub repository for tooling.
- Accessing private WordPress admin exports, backend SSH, database state,
  cookies, browser profiles, private GitHub resources, or any credentials.
- Mirroring public comments.
- Redistributing third-party media with unclear licensing.
- Choosing or changing the open-source licence for scripts.
- Running broad cleanup or deleting local state outside this repository.

Continue without asking when:

- Fetching public WordPress REST records, public canonical HTML, public media
  files, sitemap/RSS data, and public Google Sheet exports.
- Regenerating local archive content and validation reports inside this working
  tree.
